字数统计: 276 | 阅读时长 ≈ 1

1.设置管理员密码

$ sudo vi /etc/tomcat8/tomcat-users.xml //我这里是wsl_ubuntu18.04_直接通过apt的方式安装的tomcat8，和大家的目录可能不同，记的修改。

2.解除上传war包大小限制

vim 你Tomcat目录/webapps/manager/WEB-INF/web.xml
本文 为wsl有所不太：
$ sudo vi /usr/share/tomcat8-admin/manager/WEB-INF/web.xml


如图，找到大概在56行处修改，
比如要限制上传最大为500M，数字计算方法：50010241024
如图:

保存后，重启Tomcat，先停止在启动 tomcat没有重启
# service tomcat stop
# service tomcat start
上传测试，打开即可进入Tomcat Web Application Manager 主页面
网页 http://你的IP或网址:8080/manager/html/ 输入配置的用户名和密码
